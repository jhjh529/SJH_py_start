## 자료형

##### 🔥 String interpolation

> 문자열 안에서의 변수 사용

```dart
    void main() {
        var name = "seol";
        var age = "19";
        print("my name is ${name}, I'm ${age}");
    }
```

##### 🔥 List

> List\<T\> list

```dart
    void main() {
        var list = [1, 2, 3];
        print("list: ${list}");
    }
```
